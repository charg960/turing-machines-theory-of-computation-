# Theory of Computation — Turing Machines & Formal Language Models

This repository contains a collection of Turing Machine constructions, formal language models, and computational theory exercises developed through coursework and independent study in Theory of Computation.

The project explores core concepts in computability and automata theory, including:

- Turing Machine design
- Language recognition
- Decidability and computability
- State-transition systems
- Formal proofs and machine construction
- Algorithmic limitations of computation

## Repository Structure

```text
.
├── machines/        # Turing Machine definitions
├── diagrams/        # State diagrams and visual representations
├── simulations/     # Optional simulators or testing scripts
├── notes/           # Proof sketches and theoretical notes
└── README.md
```

## Example Machines

Some example constructions included in this repository may include:

- Binary increment machines
- Palindrome recognition
- Equal numbers of 0s and 1s
- Unary arithmetic operations
- String pattern recognition
- Decider and recognizer constructions

## Goals of This Repository

This repository serves as both:

1. A structured archive of computational theory work
2. A demonstration of formal reasoning and algorithmic design

The emphasis is placed on rigorous logical construction, clarity of state transitions, and the mathematical foundations of computation.

## Technical Focus

The work in this repository reflects interests in:

- Theoretical computer science
- Discrete mathematics
- Computational modeling
- Formal systems
- Mathematical logic
- Complexity and computability theory

## Notes

Most machines were originally developed as part of undergraduate Theory of Computation coursework and later refined for organization, readability, and documentation.

Additional simulations and visualization tools may be added over time.
